# CatDog-Classifier
- English Version

The CatDog image classification project was developed following my integrated learning, utilizing a publicly available cat-and-dog image dataset from Kaggle. I authored the primary code via interactive collaboration with AI, followed by manual refinement, detail adjustment, and error correction based on technical notes. The core functionality of the project is to perform binary classification on input images, outputting a corresponding label indicating whether the subject is a cat or a dog.
